# ucl_draw
A web app to simulate the draw of the Round of 16 in the UEFA Champions League.

Check it out! https://ucldraw.herokuapp.com/

## Stack

 - Python (Flask framework)
 - [Pure.css](http://purecss.io/)
 - Heroku hosted
